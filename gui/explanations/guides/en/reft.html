ReFT

Block-level residual-stream intervention (Wu et al., NeurIPS 2024). One ReFTModule per selected DiT block adds R^T · (ΔW·h + b) · scale to the block's output — an additive side-channel, not a weight delta on any Linear. Composes with any LoRA variant and lives in the same .safetensors.

Pick blocks with reft_layers (e.g. last_8, stride_2, or comma-separated indices). reft_dim controls intervention rank; reft_alpha sets effective scale via alpha / dim.

tlora_ortho_reft bundles ReFT with T-LoRA + OrthoLoRA in one training run — the LoRA half is bakeable, but the ReFT block hooks aren't, so the merged DiT loses ReFT's contribution (the merge tool warns and asks for --allow-partial).
