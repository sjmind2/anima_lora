LoRA

Classic low-rank adaptation. Adds small trainable matrices (down × up) to existing weight layers.
y = x + (x @ down @ up) × scale × multiplier
Simple, effective, and the default choice for most fine-tuning tasks. Fully bakeable into the base DiT.

Variants
• lora — default 16GB+ profile.
• lora-8gb — low-VRAM profile (block swap + offload checkpointing); pick this if you OOM on the default.
