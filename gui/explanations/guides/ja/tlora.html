T-LoRA

タイムステップ依存のランクマスキング。冪乗則スケジュールによってノイズ除去タイムステップに応じて有効 LoRA ランクが変化します:
• 高ノイズ (初期ステップ) → フルランク (最大表現力)
• 低ノイズ (後期ステップ) → 縮小ランク (min_rank まで)
構造が決定される場所に容量を集中させます。マスクは学習時のみ — 推論では設計上すべての t でフルランクを使用するため、保存された .safetensors は通常のベイク可能な LoRA です。詳細は docs/methods/timestep_mask.md を参照。

トグル: use_timestep_mask = true。

バリアント
• tlora — OrthoLoRA の上に積み重ねた T-LoRA (use_ortho = true + use_timestep_mask = true)。デフォルトのマージ可能な組み合わせ — 直交正則化付きランク-σ スケジューリング。完全にベイク可能。
• tlora-8gb — 同じアーキテクチャの低 VRAM プロファイル (勾配チェックポインティング + オフロード); デフォルトで OOM になる場合はこちらを選択してください。
