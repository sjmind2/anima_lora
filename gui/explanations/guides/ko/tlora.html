T-LoRA

타임스텝 의존 랭크 마스킹. 디노이징 타임스텝에 따라 유효 LoRA 랭크가 거듭제곱 스케줄로 변동:
• 높은 노이즈 (초기 스텝) → 전체 랭크 (최대 표현력)
• 낮은 노이즈 (후기 스텝) → 축소된 랭크 (min_rank까지)
구조가 결정되는 시점에 표현력을 집중시킵니다. 마스크는 학습 시에만 적용되며 추론은 모든 t에서 전체 랭크를 사용하므로, 저장된 .safetensors는 베이킹 가능한 일반 LoRA입니다. docs/methods/timestep_mask.md 참조.

토글: use_timestep_mask = true.

변형
• tlora — T-LoRA + OrthoLoRA 스택 (use_ortho = true + use_timestep_mask = true). 기본 베이킹 가능 조합 — σ별 랭크 스케줄링과 직교성 정규화를 함께 적용합니다. 완전히 베이킹 가능합니다.
• tlora-8gb — 동일한 아키텍처, 저VRAM 프로필 (그래디언트 체크포인팅 + 오프로드). 기본 변형에서 OOM이 발생하면 선택하세요.
